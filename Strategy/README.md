# 전략 패턴

## 개념
- 행동 패턴 중 한 가지 종류. 전략 패턴은 디자인 패턴의 꽃.
- Target 을 추상화 시킴.

## 쓰는 이유?
- 객체 자체를 의존하지 않고 추상화된 것을 의존하기 때문에, Solid 를 가능하게 함.
- 추상화 할 대상은`클래스`, `메서드` 등 다양함.

## 예시 코드 설명 
- *`Animal` 이라는 클래스를 추상화하였고,*
- *`DoorMan`이 `Animal`을 상속하는 클래스를 모두 사용할 수 있도록 하는 전략 패턴을 구현함.*
